Positioning

MANIFESTO

K-POP은 스트리밍으로 소비되지만,
팬의 애정은 소장으로 완성된다.

ONVINYL은 흩어져 있던 K-POP 바이닐을 한 곳에 모은 아카이브이자 커머스 플랫폼입니다.
음반을 단순한 상품이 아닌 기록으로 다루며,
발매 정보·한정반·컬러 디스크까지 — 팬이 진짜 궁금해하는 맥락을 함께 큐레이션합니다.

THREE PILLARS

  1. P1
    Archive first

    발매·재발매·한정반을 시간순으로 기록하는 자료성

  2. P2
    Object commerce

    ‘물성’을 보여주는 커머스 — 컬러 디스크·패키지 중심 비주얼

  3. P3
    Editorial curation

    주간 아티스트·추천처럼 잡지 같은 큐레이션 흐름

02

RESEARCH

기존 LP 쇼핑몰 분석 문제점 도출 · 레퍼런스

Competitive
Audit

국내 LP 쇼핑몰은 판매는 하지만,
맥락을 팔지 않는다.

기존 음반 커머스 4곳을 직접 구매 흐름까지 따라가며 분석했습니다.
공통적으로 상품 정보는 충실했지만,
K-POP 바이닐 특유의 한정성·컬러 변형·발매 일정을 다루는 정보 구조가 부재했고,
비주얼은 일반 쇼핑몰 템플릿에 머물러 있었습니다.

Audit Axis · 분석 항목 일반 음반몰 A 오픈마켓 B 해외 셀렉트샵 C
한정반 / 컬러반 정보 텍스트 한 줄 없음 이미지 강조
발매 / 예약 일정 목록 혼재 불명확 캘린더
아카이브 · 큐레이션 없음 없음 에디토리얼
비주얼 톤 템플릿형 정보 과밀 미니멀
ONVINYL 컬러반 비주얼 우선 · 감각적 진열 예약/디데이 명시 주간 아카이브 큐레이션

01

맥락 없는 상품 나열

발매 연대와 아티스트 흐름이 빠져 있어 팬의 탐색 경험이 단편적으로 끊겼습니다.

02

예약·발매 정보의 단절

예약 상태와 발매 일정이 분산돼 있어 출시 타이밍과 예약 흐름이 자연스럽지 않았습니다.

03

물성이 안 보이는 비주얼

레코드의 컬러 디스크와 패키지 차별점이 섬네일만으로 충분히 드러나지 않았습니다.

Reference / Moodboard — 레퍼런스 수집
Key Insight

상품 판매 중심 구조에서 벗어나 아티스트와 LP 문화를 함께 경험할 수 있는 에디토리얼 커머스 구조를 목표로 삼았습니다.
